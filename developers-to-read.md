# Developers to read

http://github.com/fast-flow/react

### Development

```shell
npm run dev
# or npm run dev:debug
npm run js
# npm run js:debug
```

### Test

```shell
npm run test
```

### Publish gh-pages
```shell
npm run gh
npm run s
npm run gh-push
```

### Publish npm

```shell
npm run npm
cd output && npm publish
```

### Release

```shell
# git push gh-pages  & npm publish
npm run release
```
